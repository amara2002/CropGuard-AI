# Quick Start - 5 Minutes to Running CropGuard

## The Fastest Way to Get Started

### 1. Download Node.js (if you don't have it)
- Go to https://nodejs.org/
- Download the LTS (Long Term Support) version
- Install it (just click Next, Next, Finish)

### 2. Download Your Project
- Download the project ZIP file from Manus
- Extract it to a folder (e.g., `C:\Users\YourName\Documents\crop_disease_detection`)

### 3. Open in VS Code
- Open VS Code
- File → Open Folder → Select your `crop_disease_detection` folder

### 4. Open Terminal in VS Code
- Press `Ctrl + ~` (or go to View → Terminal)

### 5. Run These Commands (Copy & Paste)

```bash
npm install
```

Wait for it to finish (you'll see "added X packages"), then:

```bash
npm run dev
```

### 6. Open in Browser
- You'll see: `➜  Local:   http://localhost:3000/`
- Copy that URL and paste it in your browser
- Press Enter

**Done! 🎉 Your app is now running!**

---

## What You'll See

1. **Home Page** - Click "Get Started Now"
2. **Signup Form** - Fill in your details (3 steps)
3. **Dashboard** - Upload crop images
4. **Profile Settings** - Edit your information
5. **Account Settings** - Change language and notifications

---

## If Something Goes Wrong

**Error: "npm: command not found"**
- Node.js is not installed. Download from https://nodejs.org/

**Error: "Port 3000 already in use"**
- Close other applications using port 3000
- Or stop the server and restart

**Blank page in browser**
- Press `Ctrl + Shift + R` to hard refresh
- Check the browser console (F12) for errors

---

## Stop the Server

Press `Ctrl + C` in the VS Code terminal

---

## Restart the Server

Run `npm run dev` again

---
